# S-G42-15: Evidence Sufficiency Diagnosis Routing

- **Type**: Deterministic
- **Step**: Evidence Sufficiency Gate (G42, Step 16A)
- **Batch**: P0
- **Agent**: authoring-evidence-agent

## Input
- `pre_g42_claim_evidence_candidate_matrix`
- `claim_ledger` with claim_type
- `CLAIM_TYPE_SUFFICIENCY` per-type thresholds

## Output
- Gate decision: PASS / REWORK_REQUIRED / BLOCKED
- `g42_failure_pattern_report` with failure_pattern and repair_route
- Spiral round tracking

## Decision Logic (13 failure patterns, priority-ordered)
| Priority | Pattern | Diagnosis | Repair Route |
|----------|---------|-----------|-------------|
| 1 | SOURCE_TYPE_REQUIREMENT_NOT_MET | Required source type missing | risk_gspr_mapping |
| 2 | ALLOWED_USE_BLOCKED | Evidence blocked by use restrictions | claim_decomposition |
| 3 | MISSING_DATA_BLOCKING | Required data fields absent | evidence_appraisal |
| 4 | LINKING_GAP | Claim-evidence link not established | pre_g42_linking |
| 5 | CLAIM_TYPE_MISCLASSIFICATION | Wrong claim_type assigned | claim_decomposition |
| 6 | CLAIM_SOURCE_MISMATCH | Evidence source doesn't match claim type | risk_gspr_mapping |
| 7 | SOURCE_TYPE_INAPPROPRIATE | Source type inappropriate for claim | risk_gspr_mapping |
| 8 | ENDPOINT_GAP | Endpoint missing for quantitative claim | endpoint_extraction |
| 9 | PDF_GAP | Full-text unavailable for pivotal evidence | evidence_appraisal |
| 10 | OCR_GAP | OCR quality insufficient | evidence_appraisal |
| 11 | SEMANTIC_SUPPORT_NOT_ESTABLISHED | Semantic link too weak | pre_g42_linking |
| 12 | CLAIM_OVERREACH | Claim scope exceeds evidence | claim_evidence_matrix |
| 13 | EVIDENCE_TRULY_INSUFFICIENT | Genuinely insufficient after 2 spiral rounds | query_expansion |

Per-type thresholds from `CLAIM_TYPE_SUFFICIENCY` override uniform 13-pattern standard.
Spiral: 3 rounds max. Round 3 still insufficient → BLOCKED → controlled_compromise.

## Checks
- 13 patterns all have diagnosis conditions
- Per-type thresholds applied before uniform check
- Spiral round limit enforced
- Diagnosis accuracy ≥ 80% on 10+ cases
